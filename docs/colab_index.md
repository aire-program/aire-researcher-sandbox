### How to run in Colab
1. Click a badge to open the notebook in Google Colab.
2. Run the first setup cell. It detects the Colab runtime and installs dependencies automatically.
3. Execute the remaining cells from top to bottom. All notebooks use synthetic data stored in this repository; clone the repo or upload the `data/` directory if you need to rerun generation steps inside Colab.
